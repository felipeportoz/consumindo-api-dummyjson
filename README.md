# Projeto Busca de Produtos

Atividade de desenvolvimento web com consumo de API externa.

## 🚀 Tecnologias Utilizadas

- HTML5 (Semântico)
- CSS3 (Responsivo)
- JavaScript (Assíncrono com Fetch API)
- API DummyJSON

## 🛠️ Funcionalidades e Critérios Atendidos

- Busca de produtos por ID e por texto.
- Validação de dados de entrada do usuário.
- Layout totalmente responsivo.
